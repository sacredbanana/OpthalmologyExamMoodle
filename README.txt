INTRODUCTION
This project is a set of modifications and theme for the Moodle (Modular Object-Oriented Dynamic Learning Environment) online test taking platform, written in the PHP programming language. Our system is designed to be used by Royal Australian and New Zealand College of Ophthalmology for opthamology exams.

SOURCE DIRECTORY
The source directory is organized in a way that allows it to be extracted into a web server's Moodle directory with all the files moved to the correct location.

INSTALLATION AND USAGE
Refer to INSTALL.txt for installation and the User Documentation directory for usage.
